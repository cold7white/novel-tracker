# Novel Tracker - 简历版

## 项目描述
个人小说阅读记录管理平台，帮助用户追踪阅读进度、记录心得并通过数据可视化展示阅读习惯。

## 技术栈
React 19 | TypeScript | Vite | Supabase | CSS Variables | Vercel

## 核心功能
- 双视图模式（卡片/列表）与智能筛选排序系统
- 完整的 CRUD 操作和分类管理功能
- 数据统计仪表板（环形图、条形图、折线图可视化）
- 移动端响应式设计与方向感知布局
- 本地缓存与云端数据同步
