# Novel Tracker 项目经历

## 项目描述

**Novel Tracker** - 个人小说阅读记录管理平台

一款功能完整的阅读管理应用，帮助用户追踪阅读进度、记录阅读心得，并通过丰富的数据可视化展示阅读习惯。

## 技术实现

**技术栈**：React 19 + TypeScript + Vite + Supabase

- **前端**：采用 React 19 Hooks 和 Context API 实现状态管理，原生 CSS + CSS 变量实现响应式设计
- **后端**：集成 Supabase BaaS，实现用户认证、数据存储和实时同步
- **部署**：Vercel 自动部署，支持多环境配置

**核心功能**：
- 双视图模式（卡片/列表）与智能筛选排序
- 完整的 CRUD 操作和分类管理系统
- 数据统计仪表板（环形图、条形图、折线图）
- 移动端响应式设计与方向感知布局
- 本地缓存与云端数据同步
